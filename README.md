介绍
===================================
源代码来自 https://github.com/hoang8f/android-segmented-control

> ## 改动部分
> 
> 1. 不依赖任何 drawable 文件。
> 2. 支持垂直排列布局。
> 3. 可通过 xml 配置属性，包括 mainColor, subColor, cornerRadius, strokeWidth。
